  - Cloudflare 推出基于 Code Mode 的新 MCP 服务器

  - 未来将支持 Cloudflare MCP Server Portals，组合多个 MCP 服务器

- **Markdown for Agents** (2026-02-12) ⚡ **新功能**
  - Cloudflare 网络支持实时内容转换
  - AI 系统可以通过 `Accept: text/markdown` 头直接请求 markdown 格式内容

  - 包含 Content-Signal 头，表示内容可用于 AI Training、Search 和 AI Input
  - Cloudflare Radar 现在可以跟踪 AI 机器人和爬虫对 markdown 的使用情况

## 总结

### 重要趋势

1. **MCP 和 Code Mode 成为主流**: Cloudflare 的 Code Mode 技术通过让模型编写代码而非使用多个工具，将 API 访问的 token 使用量减少 99.9%，这一模式被 Anthropic、Claude SDK 等广泛采用

2. **Markdown 格式成为 AI 代理的标准**: Cloudflare 的 Markdown for Agents 功能让 AI 代理可以直接请求 markdown 格式，减少 80% 的 token 使用，标志着 Web 正在从"为人构建"向"为代理构建"转变

3. **代理可观测性和性能优化**: LangChain Deep Agents 通过优化 harness 将 Terminal Bench 2.0 排名从 Top 30 提升至 Top 5，显示代理框架正从实验转向生产级性能优化

4. **模板化和快速开发**: LangChain 推出 Agent Builder 模板库，让开发者可以快速部署预配置的代理，降低了代理开发的门槛

5. **多代理系统崛起**: 从单代理到多代理编排是2025-2026年最重要的范式转变，CrewAI、AutoGen（转型中）、LangGraph 都在这一方向发力

6. **框架整合**: 微软退役 AutoGen 并推出统一的 Agent Framework，反映了企业级代理框架整合的趋势

7. **数据所有权受关注**: Moltbot 的快速流行和 Cloudflare Moltworker 的推出反映了用户对数据隐私和自托管的需求
